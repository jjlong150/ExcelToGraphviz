# Microsoft Edge has Blocked the Graphviz Download

The Microsoft Edge browser may blocked the download of the Graphviz installer file. 

If blocked, you will see an orange `/!\` warning icon in the download button.  
![Microsoft Edge download bar showing an orange warning icon indicating that the Graphviz installer has been blocked.](./blocked_icon.png)

## Step 1 - View Warning Message

Clicking on the button provides the reason the download was blocked.

| ![Microsoft Edge download warning dialog explaining why the Graphviz installer was blocked.](../media/94ce6a6cbad3583dec3caac5a22e9510.png) |
| ------------------------------------- |

## Step 2 - Make `More Actions` appear

Hover your mouse over the message, and a button with three dots will appear, along with a tooltip which says, `More actions`.

| ![Microsoft Edge download warning panel showing the three‑dot "More actions" button revealed on hover.](../media/b09a6c5299b24dc133e167dbbe2b20ae.png) |
| ------------------------------------- |

Click on the **`⋯`** button, and a popup menu appears. 

## Step 3 - Select `Keep`

Select `Keep` from the dropdown list.

| ![Microsoft Edge download security menu showing the "Keep" option used to allow the blocked Graphviz installer to be saved.](../media/bcf6a219b278f7796726543079756485.png) |
| ------------------------------------- |

## Step 4 - Confirm Again

Microsoft Edge will again try to dissuade you from downloading the file with a warning such as “**Make sure you trust windows_10_cmake_Release_graphviz-install-14.1.4-win32.exe before you open it**”. 

It will appear that your only choices are `Cancel` or `Delete`. To keep the file you must click the **`v`** dropdown within the Delete button.

| ![Microsoft Edge security warning showing only Cancel and Delete options, with a dropdown arrow inside the Delete button to reveal additional actions.](../media/3466d8ae66805d90080ee3083f188f1a.png) |
| ------------------------------------- |

An additional choice appears:

- `Keep anyway`

| ![Microsoft Edge download warning menu expanded to show the "Keep anyway" option for allowing the blocked Graphviz installer.](../media/efd3d173af8c782fb739f9a0698e3066.png) |
| ------------------------------------- |

Click on `Keep anyway`

## Step 5 - Open the File

Microsoft Edge downloads the file, and the installer file shows up as a download. Click on `Open file` to run the installer.

| ![Microsoft Edge download bar showing the completed Graphviz installer download with an “Open file” button to launch the installer.](../media/7fd0cb6d4e676caea950589d2c1e8cc2.png) |
| ------------------------------------- |

## Step 6 - Resume Installation

Resume the [installation steps](../install-win/)
